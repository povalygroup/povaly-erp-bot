📢 OFFICIAL DIRECTIVES GUIDE — COMPANY ANNOUNCEMENTS

🎯 PURPOSE: Official company announcements, policies, and directives from leadership.

🔐 ACCESS CONTROL
👑 Authorized Posters: Chairman, CEO, Managers, Owners
👤 All Employees: Can read and acknowledge
🚫 Employees CANNOT post messages

📝 ANNOUNCEMENT FORMAT

[DIRECTIVE] Policy/Announcement title
[EFFECTIVE_DATE] YYYY-MM-DD
[PRIORITY] LOW / MEDIUM / HIGH / CRITICAL
[CONTENT] Full announcement text
[ACTION_REQUIRED] Yes/No
[DEADLINE] YYYY-MM-DD (if action required)

Example:
[DIRECTIVE] New Work From Home Policy
[EFFECTIVE_DATE] 2026-05-01
[PRIORITY] HIGH
[CONTENT] Effective May 1st, employees can work from home 2 days per week...
[ACTION_REQUIRED] Yes
[DEADLINE] 2026-04-30

🔄 ACKNOWLEDGMENT SYSTEM

Employees acknowledge by reacting:

👍 = ACKNOWLEDGED (I read and understand)
❤️ = ACKNOWLEDGED & AGREED (I agree)
👎 = ACKNOWLEDGED & DISAGREE (I read but disagree)

🤖 AUTOMATION LOGIC

On announcement posted:
→ Bot pins message automatically
→ Sends notification to all employees
→ Tracks acknowledgment status

On 👍 reaction:
→ Mark as acknowledged
→ Store timestamp
→ Send DM confirmation

On ❤️ reaction:
→ Mark as acknowledged & agreed
→ Store timestamp
→ Send DM confirmation

On 👎 reaction:
→ Mark as acknowledged & disagreed
→ Store timestamp
→ Alert admin: "User disagreed"

⏰ ACKNOWLEDGMENT TRACKING

Admins can track acknowledgments by checking reactions on directive messages.

👍 = User acknowledged
❤️ = User agreed
👎 = User disagreed

Check message reactions to see who has acknowledged.

⚡ COMMANDS

/pin - Pin directive message

Note: Directive system is currently manual. Post announcements in the format above and use /pin to pin them.

👑 FOR LEADERSHIP/MANAGERS

WORKFLOW:
1. Compose directive
2. Post in Official Directives topic
3. Fill in all required fields
4. Set priority and deadline
5. Use /pin to pin message
6. Monitor acknowledgments via reactions
7. Follow up on non-acknowledgers
8. Track disagreements

⚠️ Important:
• Use clear language
• Set realistic deadlines
• Follow up on acknowledgments
• Address disagreements
• Document everything

👷 FOR EMPLOYEES
