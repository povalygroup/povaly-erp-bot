👷 FOR EMPLOYEES

═══════════════════════════════════════════════════════

📝 COMPLETE TASK WORKFLOW

STEP 1 - ASSIGNED → STARTED
React 👍 on task message
→ Task state: ASSIGNED → STARTED
→ Auto check-in (if first task of day)
→ Confirms you're working on it

STEP 2 - WORK IN PROGRESS
Keep 👍 reaction as reminder (optional)
→ You can add/remove 👍 freely
→ No notifications for add/remove
→ Use as personal marker

STEP 3 - COMPLETE WORK
Finish according to requirements
Check deadline reminders in DM

STEP 4 - SUBMIT FOR QA
Go to QA Review topic
Use /newqa command or manual format
→ Task state: STARTED → QA_SUBMITTED

STEP 5 - QA APPROVAL
Reviewer approves with ❤️
→ Task state: QA_SUBMITTED → APPROVED
→ You get DM notification (90s auto-delete)

STEP 6 - CONFIRM COMPLETION
React ❤️ on task message (in Task Allocation)
→ Task state: APPROVED → COMPLETED
→ You get DM confirmation (60s auto-delete)

STEP 7 - AUTOMATIC ARCHIVE
After 24 hours in COMPLETED state
→ Task state: COMPLETED → ARCHIVED
→ Task moved to Central Archive

─────────────────────────────────────────────────────

🔄 REACTION SYSTEM

👍 THUMBS UP (Flexible Usage)
✅ On ASSIGNED: Start task (ASSIGNED → STARTED)
✅ On STARTED: Allowed (no action, just marker)
✅ On QA_SUBMITTED: Allowed (no action, just marker)
✅ On REJECTED: Allowed (no action, just marker)
❌ On APPROVED: Not allowed (use ❤️ instead)
❌ On COMPLETED: Not allowed (already done)

Remove 👍 from STARTED:
→ Reverts to ASSIGNED (only if no other reactions)
→ DM notification (60s auto-delete)

❤️ HEART (Completion Confirmation)
✅ On APPROVED: Confirm completion (APPROVED → COMPLETED)
❌ On other states: Not allowed (see error message)

Remove ❤️ from COMPLETED:
→ Reverts to APPROVED
→ DM notification (60s auto-delete)

─────────────────────────────────────────────────────

💡 SMART REACTION DETECTION

If you change 👍 to another reaction (❤️, 👎, 🔥):
→ NO "unclaim" notification
→ System knows it's a reaction change, not unclaim

If you remove 👍 with no other reactions:
→ SENDS "unclaim" notification
→ System knows it's a true unclaim

This prevents spam notifications!
